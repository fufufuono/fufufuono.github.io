# I'm Blue

Jekyll source for `https://fufufuono.github.io`.

## Write posts online

1. Open `https://app.pagescms.org/`.
2. Sign in with GitHub.
3. Connect the `fufufuono/fufufuono.github.io` repository.
4. Create or edit entries in the `Posts` collection.
5. Save changes and let GitHub Pages rebuild the site.

## Analytics

Set `cloudflare_analytics_token` in `_config.yml` after creating a Cloudflare Web Analytics site. Each post has its own URL, so the analytics dashboard can show visits per post path.

Google Analytics can also be enabled by setting `google_analytics_id`.
